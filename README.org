* OpenGL Starter Code
  You may replace "my_project" with your project name or leave it as it is.

** Requirements
Make sure you have installed following extensions in VS Code:
- CMake
- CMake Tools

** Build Instruction
Go to the folder where you want to clone this repository and perform following steps:
#+begin_src bash
git clone --recursive --depth 1 https://github.com/sudiptiwari/opengl_starter.git
#+end_src
#+begin_src bash
cd opengl_starter
#+end_src
#+begin_src bash
mkdir build
#+end_src
#+begin_src bash
cd build
#+end_src
#+begin_src bash
cmake ..
#+end_src
#+begin_src bash
make
#+end_src
#+begin_src bash
cd my_project
#+end_src
#+begin_src bash
./my_project
#+end_src
